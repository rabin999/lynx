# NodeJS Sandbox
## Component based architecture for nodejs application


## Installation &nbsp;
**With [node](http://nodejs.org) [installed](http://nodejs.org/en/download):**
```sh
$ git clone https://github.com/rabin999/NodeJsSandBox
$ npm install
$ npm run dev
```
